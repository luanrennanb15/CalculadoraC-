
# Calculadora Console em C#

Este projeto é uma **calculadora simples** desenvolvida em C# para **fins didáticos**.

## Objetivo
- Treinar conceitos de **controle de fluxo** (`goto`, loops, switch-case)
- Praticar **operações matemáticas básicas** (soma, subtração, multiplicação, divisão)
- Aprender a **formatar números** e lidar com entrada de usuário

## Observações importantes
- O uso de `goto` foi **intencional** para estudo.  
- **Não é recomendado** usar `goto` em projetos reais.
- Futuras versões do projeto vão substituir `goto` por loops e métodos reutilizáveis.

## Funcionalidades
- Menu interativo
- Soma, subtração, multiplicação, divisão
- Validação de divisão por zero
- Formatação de números com casas decimais
- Espaço para salvar contas (em desenvolvimento)
